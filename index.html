Memory Mates

A native iOS application that helps caregivers better ensure their loved ones are on track with their daily responsibilities through push notification reminders. Particularly targeted for those who suffer from dementia, Alzheimer's, or other memory-related illnesses. Memory Mates removes the technical barrier of the typical reminder app by outsourcing the setup to a loved one or caregivers.

How we built it

Memory Mates was first conceptualized in Adobe Figma into wireframes that were then translated into views in a SwiftUI native IOS mobile application. Xcode among other tools such as GitHub were used for development and source control with this project showcase hosted on Github Pages. The backend behind Memory Mates is a nodejs express api connected to a Mongoose Atlus database. For sending invites to your mates, we used Twilios SMS messaging API service to send text messages to the person of your choice and provide a custom signup link just for them. Finally, we utilized Apple Push Notification service to send push notifications directly to your device in real-time to power our reminder service.
